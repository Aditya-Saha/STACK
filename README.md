# STACK
Problem Statement:
Operations on Stack.
Write a menu driven program to perform The Following
Main Menu: Stack operations
1. Using Array
2. Using Linked List
3. Exit
Sub-Menu 1: Stack Using Array
1. Push
2. Pop
3. Reverse the stack using additional stack
4. Display
5. Back
Sub-Menu 2: Stack Using Linked List
1. Push
2. Pop
3. Display
4. Back
